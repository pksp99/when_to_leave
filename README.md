# When To Leave

A project designed to help you determine the best time to leave for an online reservation, ensuring you arrive neither too early nor too late.

## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/when_to_leave.git
    cd when_to_leave
    ```

2. Create and activate a conda environment:
    ```bash
    conda env create -f environment.yml
    conda activate when_to_leave
    ```

3. Install the package in editable mode:
    ```bash
    pip install -e .
    ```

4. To run and experiment with the code, use the notebooks in the `notebooks/` directory.
